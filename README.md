# (2026) Rapid modelling of the 2026 Andes virus outbreak predicts limited transmissibility and low mainland outbreak risk

## Contents

| Directory               | Description                                                                             |
|-------------------------|-----------------------------------------------------------------------------------------|
| `data/`                 | Contains all model parameterisation files used in this study                            |
| `models/`               | Contains all calibration model outputs                                                  |
| `scripts/`              | Contains all scripts used in this study including all parameterisation and model runs   |
| `.here`                 | Empty text file used for dynamic pathing by the R here() library                        |
| `LICENCE.txt`           | CC-BY-4.0 license designation                                                           |
| `README.md`             | Markdown file for repository documentation                                              |
